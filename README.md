# Creation of a corporate social network

## Getting started

`git clone git@github.com:Bastos81/JessyBastos_7_10122020.git`

### Frontend :


cd frontend
npm install 
npm run serve


### Backend :


cd backend
npm install
nodemon server


### Database :

cd backend

npx sequelize-cli db:create
npx sequelize-cli db:migrate

http://localhost:8080/
